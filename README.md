# movie-genre-classification


🎬 Movie Genre Classification using Machine Learning
An AI-powered web application that predicts movie genres from text descriptions using Natural Language Processing (TF-IDF) and Machine Learning, deployed with Streamlit.

📌 Project Overview

This project classifies movie descriptions into genres such as:
Drama
Romance
Action
Horror
Comedy
The model learns patterns from text data and predicts the most suitable genre.

🧠 Key Features
Predict movie genre from text input
NLP-based feature extraction using TF-IDF
Real-time prediction using Streamlit
Simple and interactive user interface
Lightweight and fast machine learning model

🛠️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
NLP (TF-IDF)
Streamlit

🚀 How to Run

Install dependencies using requirements.txt and run the Streamlit app using:
pip install -r requirements.txt
streamlit run app.py

🎯 Example

Input:
A young couple falls in love but faces emotional struggles.
Output:
Romance / Drama
